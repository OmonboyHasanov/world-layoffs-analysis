# Global Layoffs Data Analysis (SQL)

📊 **Project Overview:**  
This project analyzes a global layoffs dataset with 2,362 records from multiple companies and countries. The goal is to clean the raw data and explore patterns in layoffs across industries, countries, and time periods using SQL.

---

## 🔹 Key Tasks

- **Data Cleaning:**  
  - Removed duplicates and handled missing values  
  - Standardized formats for company names, countries, and dates  
  - Prepared data for analysis  

- **Exploratory Data Analysis (EDA):**  
  - Analyzed layoffs by company, country, and industry  
  - Identified trends and patterns in layoffs over time  
  - Built SQL queries to summarize insights for reporting  

---

## 🔹 Key Insights

- Tech companies had the highest number of layoffs  
- Layoffs peaked during specific economic periods  
- Certain countries were more affected than others
